About Texera Hub

The Texera Hub is an online platform for community users with various backgrounds to learn and practice data science, AI, and machine learning.

It also allows users to share their datasets and analyze workflows.

Key Features

  • Supporting no-code/low-code data science and AI/ML using GUI-based workflows
  • Real-time collaboration with shared editing and shared execution of the same workflow by multiple users
  • Allow user-defined functions (UDFs) in languages such as Python, R, Java, and Scala
  • Debugging
  • Parallel computing
  • Commenting
  • Version control
  • Access control of datasets and workflows
