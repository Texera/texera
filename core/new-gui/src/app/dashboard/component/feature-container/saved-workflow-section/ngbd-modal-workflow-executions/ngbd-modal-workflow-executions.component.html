Execution results of this workflow

We support the following search criteria:
  • Search by Execution Name: executionName
  • Search by Execution User: user:John
  • Search by Execution Starting Time: sTime:MM-DD-YYYY
  • Search by Execution Last Status Updated Time: uTime:MM-DD-YYYY
  • Search by Execution Status: status:Initializing/Running/Paused/Completed/Aborted
For any execution name and user name, if the name contains space, using double quotes to enclose the name is required.
For starting time and last status updated time, "before" and "after" searched time is also allowed.
Before syntax: sTime:<MM-DD-YYYY; After syntax: sTime:>MM-DD-YYYY
Example: "Untitled Execution" user:John sTime:07/13/2022 uTime:07/18/2022
Meaning: Search for the execution with name Untitled Execution and run by user called John starting on 07/13/2022 and last status updated time is 07/18/2022.
